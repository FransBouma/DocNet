Acknowledgements

This application wouldn't be possible without the work of others. The (likely incomplete) list below contains the work Docnet is based on / builds upon.

  • MkDocs. Docnet borrows a great deal from MkDocs: the theme css Docnet uses is based on a cleaned up version of their Readthedocs theme, as well as the javascript based search is from MkDocs.
  • MarkdownSharp. The markdown parser is an extended version of the MarkdownSharp parser from StackExchange with the extensions added by Kyle Gobel.
  • Json.NET. The reading/writing of json files is important for Docnet and it uses Json.NET for that.
Made with DocNet. Get your copy at: https://github.com/FransBouma/DocNet.
